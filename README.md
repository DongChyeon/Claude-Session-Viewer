# Claude Session Viewer

Claude Code의 대화 세션을 탐색하고, 검색하고, 터미널에서 재개할 수 있는 데스크톱 앱입니다.

---

## 설치

### macOS (.dmg)

1. `claude-session-viewer-1.0.0.dmg` 파일을 다운로드합니다.
2. DMG 파일을 열고 **Claude Session Viewer** 아이콘을 **Applications** 폴더로 드래그합니다.
3. 처음 실행 시 macOS 보안 경고가 뜨면 아래 방법 중 하나를 사용하세요.

   **방법 A — 우클릭으로 열기 (권장)**
   - Finder에서 앱 아이콘을 **우클릭(또는 Control+클릭)** → **열기** → 대화상자에서 **열기** 클릭

   **방법 B — 터미널 명령**
   ```bash
   xattr -cr "/Applications/Claude Session Viewer.app"
   ```

### Windows (.exe)

1. `claude-session-viewer-1.0.0-setup.exe` 파일을 다운로드합니다.
2. 설치 파일을 실행하고 안내에 따라 설치합니다.
3. 설치 완료 후 바탕화면 또는 시작 메뉴에서 **Claude Session Viewer**를 실행합니다.

> Windows SmartScreen 경고가 뜨면 **추가 정보** → **실행** 을 클릭합니다.

---

## 사전 요구 사항

- [Claude Code CLI](https://claude.ai/code) 가 설치되어 있어야 합니다.
- Claude Code가 `~/.claude/projects/` 에 세션 파일을 저장해야 합니다. (기본 설정이면 자동으로 저장됩니다.)
- **터미널에서 재개** 기능을 사용하려면 터미널에서 `claude` 명령이 실행 가능해야 합니다.

---

## 기능 소개

### 세션 탐색

3개의 패널로 Claude Code 세션을 탐색합니다.

```
┌──────────────┬──────────────┬──────────────────────────────┐
│   Projects   │   Sessions   │       Conversation           │
│              │              │                              │
│ 프로젝트 목록  │  세션 목록    │  선택한 세션의 대화 내용       │
│  (디렉터리)   │  (날짜/미리보기) │  (마크다운 렌더링)           │
└──────────────┴──────────────┴──────────────────────────────┘
```

- 각 패널의 경계를 드래그해서 너비를 조절할 수 있습니다.
- 대화 내용은 마크다운 형식으로 렌더링됩니다 (코드 블록, 표, 목록 등).

---

### 세션 내 검색 (`⌘F` / `Ctrl+F`)

현재 열려 있는 세션에서 키워드를 검색합니다.

- 검색어에 해당하는 부분이 하이라이트됩니다.
- `Enter` / `↓` 버튼으로 다음 결과, `Shift+Enter` / `↑` 버튼으로 이전 결과로 이동합니다.
- 해당 메시지로 자동 스크롤됩니다.
- `Esc` 또는 `✕` 버튼으로 검색을 닫습니다.

---

### 전체 검색 (`⌘K` / `Ctrl+K`)

모든 프로젝트와 세션을 통틀어 키워드로 검색합니다.

- 검색 결과에서 항목을 클릭하면 해당 세션과 메시지로 바로 이동합니다.

---

### HTML로 내보내기

현재 세션을 단독 실행 가능한 HTML 파일로 저장합니다.

- 검색바의 **↓ HTML** 버튼을 클릭합니다.
- 저장 위치를 선택하면 파일이 저장되고 브라우저에서 자동으로 열립니다.
- 저장된 HTML 파일은 인터넷 연결 없이도 대화를 볼 수 있습니다.

---

### 터미널에서 세션 재개

선택한 세션을 터미널에서 바로 이어서 대화할 수 있습니다.

1. 세션을 선택합니다.
2. 검색바 우측의 드롭다운에서 사용할 터미널을 선택합니다.
3. **▶ 재개** 버튼을 클릭합니다.
4. 선택한 터미널이 열리고 `claude --resume <세션ID>` 명령이 자동으로 실행됩니다.

**지원 터미널**

| 운영체제 | 터미널 |
|---------|--------|
| macOS | Terminal.app, iTerm2, Warp, Ghostty |
| Windows | Windows Terminal, cmd.exe, PowerShell |
| Linux | GNOME Terminal, Konsole, Kitty, xterm |

선택한 터미널은 기억되어 다음 실행 시에도 유지됩니다.

---

### 파일 자동 감지

Claude Code로 새 세션이 생성되거나 대화가 업데이트되면 앱이 자동으로 목록을 갱신합니다.

- 하단 상태바의 **● 파일 감시 중** 표시로 동작 여부를 확인할 수 있습니다.

## 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `⌘K` / `Ctrl+K` | 전체 검색 열기 |
| `⌘F` / `Ctrl+F` | 세션 내 검색 포커스 |
| `Enter` | 다음 검색 결과 |
| `Shift+Enter` | 이전 검색 결과 |
| `Esc` | 검색 닫기 |
